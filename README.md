This repository contains all of the information necessary to install and run Pokemon IronMon Kaizo Fire Red MIX n MAX
The .xlxs file contains the full set of rules required to properly play the game mode.
The .rnqs file provides the settings file necessary to randomize your seed appropriately.

Instructions for installation (Please note, this installation guide assumes an already-installed IronMon Bundle (IAteYourPie) and Max Extension Bundle (ChampRed)

Step 1: Launch Bizhawk

Step 2: Navigate to File > Open Rom

Step 3: Load your patched Mix n Max Rom

Step 4: Load the Ironmon-Tracker

Step 5: In the Tracker settings, navigate to Extensions, and make sure the MAX Extension is turned on, and all other extensions are off

Step 6: In the Tracker settings, navigate to New Runs, and create a new Profile

Step 7: In the Rom Selection line, select your patched Mix n Max Rom (same file you loaded in step 3)

Step 8: In the Randomizer Selection line, select the "upr.jar" file that came with the Max Extension Bundle

Step 9: In the RNQS Selection line, select the "Mix n Max.rnqs" file found in this repository

Step 10: Save your profile, select it, and create a new game
